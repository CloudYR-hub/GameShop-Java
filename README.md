# 🎮 GameShop Java GUI

A Java Swing desktop application for managing a game store inventory, built with a custom dynamic array data structure.

---

## ✨ Features

- 📂 **Load Games from File** – Import a game list from a `.txt` file
- ➕ **Add Game** – Add new games to the store inventory
- 📋 **Display Games** – View all games with their genre and price
- 🔍 **Search Game** – Search for a specific game by name
- ❌ **Unset Game** – Remove a game by index
- 🎭 **Remove by Genre** – Bulk remove games matching a specific genre
- 📊 **Game Analysis** – Analyze inventory data

---

## 🛠️ Technologies Used

- **Java 23**
- **Java Swing** (JFrame, JPanel, JButton)
- **Custom Dynamic Array** – built from scratch without Java's built-in ArrayList

---

## 🚀 How to Run

1. Clone the repository:
```bash
   git clone https://github.com/CloudYR-hub/GameShop-Java.git
```
2. Open in **IntelliJ IDEA**
3. Make sure **JDK 23** is installed
4. Run `GameShopGUI.java`
